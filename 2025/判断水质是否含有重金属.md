# 判断水质是否含有重金属

> 日期检查

```
1、看，用透明杯子半杯水，在光线充足的地方观察是否悬浮物的细微物质，静置半个小时后观察杯子底部是否有沉淀物；如果有沉淀物，水中悬浮物杂质超标

2、闻，接好的自来水闻是否有漂白粉的味道；如果能闻到，说明水中余氯超标

3、尝，在和烧开的白开水时，如果水的味道不甜，反而有一股漂白水味，这也可能是漂白粉超标的现象

4、查，检查家里的水壶内壁或者底部，如果发现水壶长着结有块状的杂质，说明自来水中含有水垢的成分

5、购买“水中重金属铅胶体金检测卡”对水质进行快速检测

6、如果没太大把握，买个静置水桶，静置它一天一夜或者更长的时间，这样可以极大的降低风险（在没有对应的选择情况下）

7、花点成本，买个蒸馏器，和蒸馏水（也就是前面几点都可以不用考虑）

```

> 专业检测

```
1、采取样本，送到专业检测水质机构进行专业检测，成本高，而且还要定期检测
2、如果不太想花钱，或者想降低生活成本，建议买个蒸馏器进行蒸馏
```
